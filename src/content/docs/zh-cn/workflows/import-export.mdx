工作流支持导入导出功能，用于分享、备份和迁移工作流。

## 导出工作流

在工作流列表页或者详情页，可以到导出工作流。导出时可选导出环境变量。导出格式为`.json`。

<Aside type="caution" title="环境变量">
环境变量常用于存储 API 密钥、密码等敏感信息。谨慎操作，切勿泄漏。
</Aside>

## 导入工作流

在工作流列表页，可以点击`导入`按钮，选择工作流的`.json`文件，预览内容，确认无误即可导入。
